The ArtHouse MovieAPI

Objective

To build the server-side component of a “ArtHouse Movies” web application. The web application will provide users with access to information about different ArtHouse Movies, Directors, and Genres. Users will be able to sign up, update their username, and create a list of their favorite movies.

Intro

A web application that will provide users with access to information about different movies, directors, and genres within ArtHouse flicks . Users will be able to sign up, update their personal information, and create a list of their favorite ArtHouse movies.
